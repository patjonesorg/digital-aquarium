# Digital Aquarium - Subsidiary AI Pages

This repository contains the standalone mobile-primary subpages for the specialized AI model instances deployed under the **Digital Aquarium** portal framework.

These files power the individual routing interfaces linked from the primary landing page (`aquarium.patjones.org`).
